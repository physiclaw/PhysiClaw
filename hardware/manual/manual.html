INTRODUCTION PhysiClaw.ai

What you are building

PhysiClaw.ai is a CoreXY motion platform that drives a capacitive stylus across a phone or tablet screen — the robot doesn't know it's holding a stylus, it just executes coordinates, and any app can be driven by code. This manual walks the build from bare extrusions through to a finished tap-test.

Tools you'll need

2 mm & 2.5 mm hex keys, M3 nut driver, small adjustable wrench, soldering iron (for heat-set inserts), small ruler / calipers.

How to read each section

Sections follow build order. Each section's pages list every render for that family in alphabetic filename order — _assembled first, then _exploded.

File naming

Each step has two views: _exploded shows the install motion (gaps, ghosts); _assembled shows the finished state. Camera index _cam0 by default.

Source of truth

The renders are generated from the build123d procedure files under hardware/assembly/procedures/. If a diagram and the code disagree, the code wins — rebuild the renders.

Hardware bag

Each step lists exactly the fasteners it consumes (see the BOM at hardware/output/bom/). Pull the bag for the step you're about to do, finish it, then move on.

If something looks wrong

Compare the assembled view ( _assembled_cam0.svg ) against your physical state at the end of each step. Backtracking now is cheaper than backtracking later.

HARDWARE — REFERENCE PhysiClaw.ai

Socket head cap screw (SHCS)

Cylindrical head, hex drive. The default fastener for PhysiClaw.ai — used wherever the head can sit above the surface.

ISO 4762 / DIN 912

Button head cap screw (BHCS)

Domed head, hex drive. Used on surfaces where a lower profile matters but a flush head isn't required.

ISO 7380-1

Extrusion T-nut

Drops into a 2020 / 2040 aluminum extrusion channel and locks under a screw head. Every frame-attached part rides on these.

Hex nut

Standard M3 hex nut, paired with a screw to clamp parts together where a T-nut won't fit.

ISO 4032 / DIN 934

MGN9H linear rail + slider

9 mm-wide miniature linear guide. Two rails run the Y axis, two more carry the X beam. Mounting pitch is 16 × 15 mm — don't substitute.

GT2 belt & pulley

2 mm-pitch open-loop belt, clamped at each end of the moving stage. The pulleys are 20-tooth; one per stepper, plus smooth idlers.
