# Open App via Spotlight

**Argument:** App name (e.g., "美团", "WeChat", "Safari").

## Steps

1. Send app name to clipboard
2. Go to home screen
3. Swipe down to open Spotlight
4. Long press the search field, tap Paste
5. Tap the app icon in search results
